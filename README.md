# activity_creator

This repository stores reusable instructions for generating polished, Overleaf-ready mathematics packet LaTeX documents.

## Master prompt

Use the master prompt at:

- `prompts/math-packet-creation-rules.md`

It defines two packet types:

1. Grade 9 **Active Learning** packets
2. DP **Nuanced Analysis** packets

and includes required formatting, wording, layout, graphing, spacing, command-term, and solution-markscheme rules.
